# 🚀 Member Onboarding

> T5 approval required | Agent trigger: !onboard [name] [tier]

## Workflow
1. T5 initiates: !onboard "Name" tier:1
2. Agent collects: legal name, PRO, IPI/CAE, contact, language, Google account
3. Agent creates: MEMBERS.md entry + members.json entry
4. T1 only: creates artists/[slug]/ stub + PRO.md
5. T5 provisions: Drive folder, Monday.com, SimpleTix, WhatsApp bot
6. Agent sends welcome in member's preferred language
7. Agent logs in agent/MEMORY.md

## Artist Steps (T1)
- [ ] Create artists/[slug]/ARTIST.md
- [ ] Create artists/[slug]/PRO.md
- [ ] Register all works on BMI + ASCAP
- [ ] Add SAYCE (Ecuador market)
- [ ] Set up Songtrust (global mechanical)
- [ ] Create Google Drive folder structure
- [ ] Create Monday.com release board

## Offboarding
1. T5 confirms
2. Agent sets status: inactive in members.json
3. Drive access revoked, Monday/WhatsApp removed
4. Final royalty statement archived
5. Logged in agent/MEMORY.md
