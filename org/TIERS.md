# 👥 Tercer Mundo — Tier System

## Tier 1 — Artist 🎤
- ✅ Own streams, royalties, PRO balances
- ✅ Own release catalog (ISRC, ISWC)
- ✅ PRO portal links: BMI, ASCAP, SAYCE, Songtrust
- ❌ Other artists data, org revenue, member roster

## Tier 2 — Producer / Collaborator 🎛️
- ✅ Credits + master split sheets
- ✅ Mechanical royalties, neighboring rights
- ✅ Submit beats, stems, invoices
- ❌ Booking data, org financials

## Tier 3 — Manager 📋
- ✅ Read all artist folders
- ✅ Booking pipeline — create, approve, reject
- ✅ Revenue reports per artist
- ❌ Org P&L, member onboarding

## Tier 4 — A&R / Label Ops 🎯
- ✅ Release pipeline + catalog metadata
- ✅ DSP submissions (DistroKid, TuneCore, UnitedMasters)
- ✅ ISWC registration queue
- ❌ Member management, exec financials

## Tier 5 — Executive / Owner 👑
- ✅ Full org — all tiers, all data, all agent controls
- ✅ Onboard/offboard members, configure tiers
- ✅ Org P&L, API key management
